# 🎬 Netflix Clone

A simple **Netflix Clone** built using **HTML5** and **CSS3**. This project recreates the Netflix landing page with a responsive layout and modern design, helping strengthen frontend development skills.

## 🚀 Features

* Netflix-inspired homepage
* Responsive design for different screen sizes
* Hero banner with background image
* Navigation bar
* Movie and TV show sections
* Clean and modern user interface
* Hover effects using CSS

## 🛠️ Technologies Used

* HTML5
* CSS3

## 📂 Project Structure

```text
Netflix-Clone/
│── index.html
│── style.css
│── images/
│── README.md
```

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/netflix-clone.git
```

2. Open the project folder.

3. Double-click `index.html` or open it using **VS Code Live Server**.
